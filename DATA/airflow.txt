## Apache Airflow  (ETL/ELT/,  Business Operations, MLOps, ...)

* allows you to define almost any workflow in Python code, no matter how complex.

* Airflow Core includes webserver, scheduler, CLI, ...
* Airflow CTL: command-line to interact with REST API.
* Task SDK: Allows to define Directed Acyclic Graphs (DAGs) in python, executing tasks
  in isolated subprocesses.
* Airflow providers include:
