## Perspective: C++/Rust to WebAssembly RT Data visualization 

.. despite the maturity of the Python ecosystem, few data visualization tools
 are capable of dealing with large amounts of streaming data.  Even fewer are
able to perform live aggregations, sorting, and filtering on top of this data.

Perspective interactive analytics and data visualization component, is especially
well-suited for large and/or streaming datasets.

- It is written in C++ and Rust with bindings to both Python and WebAssembly,

- It comes with a variety of visualization plugins, including a datagrid and various charts.
  as well as a Jupyter widget to iterate quickly.

...  We will start with a simple FastAPI-based website and some static data. 
In a few lines of code, we will have the website up and running.

### Perspective Core features

- pivoting.
- sorting.
- filtering.
- visualization plugins.
- cross-filtering (using one table as a filter on other tables).
- computed columns.
