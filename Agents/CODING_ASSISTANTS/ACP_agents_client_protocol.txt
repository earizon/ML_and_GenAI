# Agent Client Protocol (ACP)

* <https://github.com/agentclientprotocol/agent-client-protocol>


* Sort of standard protocol for let Agents interact with software 
  development IDEs.
* Any IDE will ACP support will automatically get integrated with any Agent
  supporting this same protocol.


## ACP Registry

* <https://agentclientprotocol.com/get-started/registry>

"The easiest way for developers to distribute and/or find and install ACP-compatible agents."

A Curated set of agents, including only the ones that support authentication (2026-04):
- GitHub Copilot
- Auggie CLI software agent
- Claude Agent
- Cline
- Mistral Vibe
- (OpenAI) Codex CLI
- (LangChain) DeepAgents (general purpose and coding agent)
- Gemini CLI
- Junie (JetBrains Coding Agent)
- Kimi CLI (Moonshot AI)
- Qwen Code (Alibaba)
- OpenCode
- ...
